About Me

Hi! I'm Tracy and I love to build, design and create applications that streamline tasks.
A little bit about my background.
I've been in finance/compliance for the past several years. My first interaction with code was a few years back when Apple first announced the Swift Playgrounds app. From there, I wanted to extrapolate the larger impact of code from a game setting to real world applications. Since then, I've learn to program in Python and Swift and have developed 4+ iOS apps, 2 of which are deployed on the App Store. It still amazes me what a few lines of code can achieve. I look forward to continue learning, solving more complex problems and building better applications that will positively impact the lives of users around the world.
